# Simple Editor

一个用 Rust 编写的终端文本编辑器，支持文件读写、文字编辑、查找替换等常用功能。

## 环境要求

- Rust 1.70+
- Windows / Linux / macOS

## 编译与运行

```bash
# 编译
cargo build --release

# 新建空白文件
cargo run --release

# 打开已有文件
cargo run --release -- filename.txt
```

编译产物位于 `target/release/simple-editor`。

## 快捷键

### 文件操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 保存文件（新文件会提示输入文件名） |
| `Ctrl+N` | 新建文件（有未保存修改时需按两次确认） |
| `Ctrl+Q` | 退出（有未保存修改时需按两次确认） |

### 编辑操作

| 快捷键 | 功能 |
|--------|------|
| 字符键 | 输入文字 |
| `Enter` | 换行 |
| `Backspace` | 向前删除 |
| `Delete` | 向后删除 |
| `Tab` | 插入 4 个空格 |

### 光标移动

| 快捷键 | 功能 |
|--------|------|
| `↑ ↓ ← →` | 移动光标 |
| `Home` | 跳到行首 |
| `End` | 跳到行尾 |
| `PageUp` | 上翻页 |
| `PageDown` | 下翻页 |

### 选择、复制、粘贴

| 快捷键 | 功能 |
|--------|------|
| `Shift + 方向键` | 选择文本 |
| `Ctrl+A` | 全选 |
| `Ctrl+C` | 复制选中内容到系统剪贴板 |
| `Ctrl+V` | 从系统剪贴板粘贴 |

### 查找与替换

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+F` | 查找 |
| `F3` | 跳到下一个匹配项 |
| `Shift+F3` | 跳到上一个匹配项 |
| `Ctrl+H` | 替换（输入查找文本 → 输入替换文本 → 全部替换） |
| `Esc` | 取消选择 / 清除搜索高亮 |

## 界面说明

编辑器界面分为三个区域：

- **编辑区**：显示文件内容，空行左侧以 `~` 标记（类似 Vim）
- **状态栏**：底部第二行，显示文件名、修改状态、光标行列号
- **消息栏**：底部第一行，显示操作提示和状态信息

查找匹配项以黄色背景高亮，选中文本以青色背景高亮。

## 项目结构

```
src/
├── main.rs       # 入口，解析命令行参数
├── editor.rs     # 主编辑器逻辑（事件循环、渲染、编辑操作）
├── row.rs        # 文本行数据结构
├── terminal.rs   # 终端初始化与输入输出封装
└── find.rs       # 查找与替换状态管理
```
