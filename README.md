# Simple Editor

- 本项目所有（100%）代码均由 AI 生成。运行前请自行审查，任何后果概不负责。
- 一个用 Rust 编写的终端文本编辑器，支持文件读写、文字编辑、查找替换、语法高亮、鼠标操作等常用功能。

## 环境要求

- Rust 1.70+
- Windows / Linux / macOS

## 编译与运行

```bash
# 编译
cargo build --release

# 新建空白文件
cargo run --release

# 打开已有文件
cargo run --release -- filename.txt
```

编译产物位于 `target/release/simple-editor`。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 保存文件（新文件会提示输入文件名） |
| `Ctrl+N` | 新建文件（有未保存修改时需按两次确认） |
| `Ctrl+Q` | 退出（有未保存修改时需按两次确认） |

## 界面说明

编辑器界面分为三个区域：

- **编辑区**：显示文件内容，空行左侧以 `~` 标记（类似 Vim）
- **状态栏**：底部第二行，显示文件名、修改状态、光标行列号
- **消息栏**：底部第一行，显示操作提示和状态信息

查找匹配项以黄色背景高亮，选中文本以青色背景高亮。

支持根据文件扩展名自动进行语法高亮，覆盖 Rust、Python、C/C++、JavaScript、TypeScript、Java、Go、Shell、JSON、TOML、HTML、CSS 等 13 种语言，包括关键字、类型、字符串、注释和数字的高亮。

## 特性

- **鼠标支持**：点击定位光标、拖拽选择文本、滚轮滚动
- **文件格式保留**：自动检测并保留文件的换行符风格（LF/CRLF）和尾部换行符
- **撤销/重做**：支持最多 1000 步操作历史的撤销和重做，包括替换全部操作的重做
- **性能优化**：批量渲染、事件驱动刷新，大文件编辑更流畅
- **终端自适应**：窗口缩放时自动重新布局

## 项目结构

```
src/
├── main.rs       # 入口，解析命令行参数
├── editor.rs     # 主编辑器逻辑（事件循环、渲染、编辑操作）
├── row.rs        # 文本行数据结构
├── terminal.rs   # 终端初始化与输入输出封装
├── find.rs       # 查找与替换状态管理
├── syntax.rs     # 语法高亮引擎
└── undo.rs       # 撤销/重做历史记录
```
